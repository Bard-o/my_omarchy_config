# My Omarchy Config

> A collection of personal customizations for [Omarchy](https://omarchy.org) — an Arch Linux distribution with Hyprland pre-configured for daily use.

## Why This Repo?

Omarchy comes out of the box with a polished, efficient desktop environment. But the best setups are personal. This repo showcases custom additions that make my Omarchy feel more like home — and hopefully inspires yours too.
